# blog-api
A simple Node.js blog API with CRUD operations, JWT authentication,input validation and basic error handling. Built with Express.js and mongoose.
# Blog API

A RESTful API for blogging platform with authentication and full CRUD operations.

## Features
- ✅ JWT Authentication
- ✅ CRUD Operations 
- ✅  registration & login
- ✅ Error handling middleware
- ✅ validation

## Tech Stack
- Node.js
- Express.js
- MongoDB
- JWT
- Mongoose
- joi
